# FitBodyAtlas - India's Complete Fitness Platform

A comprehensive fitness knowledge platform built with Next.js, offering 1000+ exercises, personalized workouts, Indian diet plans, nutrition guides, and interactive fitness calculators.

## 🎯 Mission

To empower every Indian to achieve their fitness goals through accurate, science-based fitness information specifically designed for Indian athletes and fitness enthusiasts.

## ✨ Key Features

### 📚 Content & Resources
- **1000+ Exercise Database** - Comprehensive exercise library with detailed instructions
- **Body-Part Workouts** - Chest, Back, Legs, Shoulders, Arms, Core routines
- **Veg & Non-Veg Diet Plans** - Indian-specific nutrition for all goals
- **Workout Programs** - Push/Pull/Legs, Upper/Lower, Full Body splits
- **Nutrition Guides** - Pre/Post-workout, protein foods, supplements
- **Interactive Tools** - Calorie, Protein, BMI, Macro calculators

### 🚀 Platform Features
- Modern, fully responsive design
- Mobile-first approach
- SEO-optimized for Google rankings
- Fast performance (Core Web Vitals)
- Trust & credibility pages
- Email capture & lead generation
- Success testimonials & transformations
- 24/7 accessible

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone repository
git clone https://github.com/yourusername/fitbodyatlas.git
cd fitbodyatlas

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Production Build

```bash
npm run build
npm run start
```
